## 🚀 Deploy to Vercel (Step-by-Step)

### 1. Download & Set Up Locally
1. Download the project files
2. Open your terminal and navigate to the folder:
   ```bash
   cd dinq-digital
   ```

### 2. Push to GitHub
```bash
git init
git add .
git commit -m "Initial commit — Dinq Digital website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/dinq-digital.git
git push -u origin main
```

### 3. Deploy on Vercel
1. Go to [vercel.com](https://vercel.com) and sign up / log in (use GitHub)
2. Click **"Add New Project"**
3. Import your `dinq-digital` GitHub repository
4. Vercel auto-detects the static site — click **"Deploy"**
5. ✅ Your site is live in ~30 seconds!

### 4. Custom Domain (Optional)
In Vercel dashboard → Settings → Domains → add `dinqdigital.com`
